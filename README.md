# 搜索引擎切换器 / Search Engine Switcher

## 中文介绍

这个用户脚本会在任何搜索页面或支持搜索的网站显示一个悬浮的搜索引擎切换器，让你可以快速在不同搜索引擎之间切换相同关键词，并管理自定义搜索引擎。

### 功能
- **多搜索引擎切换**：内置支持 Google、Bing、YouTube、知乎、哔哩哔哩等流行搜索引擎。
- **自定义搜索引擎**：可以轻松添加自己的搜索引擎，并在界面中管理它们。
- **自动收起**：悬浮面板在不使用时自动收起，保持页面整洁。
- **一键添加当前网站**：访问识别为搜索页面时，可快速将其保存为自定义搜索引擎。
- **通用搜索页面支持**：即使是未被自动识别的网站，只要有通用查询参数也能显示切换器。

### 使用要求
- **Tampermonkey / Violentmonkey / Greasemonkey**：需要在浏览器中安装其中一个用户脚本管理器。

### 安装与使用
1. 在浏览器中安装用户脚本管理器（Tampermonkey、Violentmonkey 或 Greasemonkey）。
2. 点击脚本的 **Install** 按钮（或复制 `.user.js` 文件内容到管理器中）。
3. 打开任意支持的搜索页面，悬浮搜索引擎切换器会自动出现。
4. 点击 **齿轮图标** 管理内置和自定义搜索引擎。
5. 点击 **➕ 添加当前网站** 可以快速保存当前搜索页面为自定义搜索引擎。

### 注意事项
- 脚本会自动识别许多流行搜索引擎，但你也可以手动添加任何支持查询参数的网站。
- 自定义搜索引擎会本地保存，并在刷新页面后仍然存在。

## English Introduction

This userscript displays a floating search engine switcher on any search page or websites that support search. It allows you to quickly switch the same keyword between different search engines and manage your custom search engines.

### Features
- **Switch Between Multiple Search Engines**: Built-in support for popular engines like Google, Bing, YouTube, Zhihu, Bilibili, and more.
- **Custom Search Engines**: Easily add your own search engines and manage them directly from the interface.
- **Automatic Collapse**: The floating panel automatically collapses when not in use to stay unobtrusive.
- **One-Click Add Current Site**: When visiting a recognized search page, you can quickly save it as a new custom search engine.
- **Visibility on Any Search Page**: Even for sites without built-in recognition, the script can show the switcher for generic search query pages.

### Requirements
- **Tampermonkey / Violentmonkey / Greasemonkey**: Install one of these userscript managers in your browser to use this script.

### Installation & Usage
1. Install a userscript manager (Tampermonkey, Violentmonkey, or Greasemonkey) in your browser.
2. Click the script's **Install** button on [Greasy Fork](https://greasyfork.org/) or copy the `.user.js` code into your manager.
3. Visit any supported search page; the floating search engine switcher will appear automatically.
4. Click the **gear icon** to manage built-in and custom search engines.
5. Click **➕ Add Current Site** to quickly save the current search page as a custom engine.

### Notes
- The script automatically detects many popular search engines, but you can manually add any site that supports query parameters.
- Custom engines are saved locally and persist across page loads.
