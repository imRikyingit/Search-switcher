# 搜索引擎切换器 / Search Engine Switcher

---

## 中文介绍

这个用户脚本会在任何搜索页面或支持搜索的网站显示一个悬浮的搜索引擎切换器，让你可以快速在不同搜索引擎之间切换相同关键词，并管理自定义搜索引擎。

### 功能 Features
- **多搜索引擎切换**  
  内置支持 Google、Bing、YouTube、知乎、哔哩哔哩等流行搜索引擎。
- **自定义搜索引擎**  
  可轻松添加自己的搜索引擎，并在界面中管理它们。
- **自动收起**  
  悬浮面板在不使用时自动收起，保持页面整洁。
- **一键添加当前页面**  
  当访问被识别为搜索页面时，可以直接将其保存为自定义搜索引擎。
- **通用搜索页面支持**  
  即使未被自动识别的网站，只要有通用查询参数，也能显示切换器。

### 安装要求 / Requirements
- **浏览器用户脚本管理器**  
  Tampermonkey / Violentmonkey / Greasemonkey  
- **获取脚本**  
  可在 [Greasy Fork 搜索引擎切换器](https://greasyfork.org/zh-CN/scripts/564560-%E6%90%9C%E7%B4%A2%E5%BC%95%E6%93%8E%E5%88%87%E6%8D%A2%E5%99%A8) 下载或安装。

### 安装与使用 Installation & Usage
1. 安装浏览器用户脚本管理器（Tampermonkey、Violentmonkey 或 Greasemonkey）。
2. 点击脚本的 **Install** 按钮，或将 `.user.js` 内容导入管理器。
3. 打开任意支持的搜索页面，悬浮搜索引擎切换器会自动显示。
4. 点击 **齿轮图标** 管理内置与自定义搜索引擎。
5. 点击 **➕ 添加当前页面** 可将当前搜索页面快速保存为自定义搜索引擎。

### 注意事项 Notes
- 脚本会自动识别许多流行搜索引擎，也可以手动添加任何支持查询参数的网站。
- 自定义搜索引擎本地保存，刷新页面后仍然有效。

**有问题请在 Pull Requests 里提交。**

## English Introduction

This userscript displays a floating search engine switcher on any search page or websites that support search. It allows you to quickly switch the same keyword between different search engines and manage your custom search engines.

### Features
- **Switch Between Multiple Search Engines**  
  Built-in support for popular engines like Google, Bing, YouTube, Zhihu, Bilibili, and more.
- **Custom Search Engines**  
  Easily add your own search engines and manage them from the interface.
- **Automatic Collapse**  
  The floating panel automatically collapses when not in use to stay unobtrusive.
- **One-Click Add Current Site**  
  When visiting a recognized search page, you can quickly save it as a new custom search engine.
- **Visibility on Any Search Page**  
  Even for sites without built-in recognition, the script can show the switcher for generic search query pages.

### Requirements / 安装要求
- **Browser Userscript Manager**  
  Tampermonkey / Violentmonkey / Greasemonkey  
- **Get the Script**  
  You can download or install it from [Greasy Fork Search Engine Switcher](https://greasyfork.org/zh-CN/scripts/564560-%E6%90%9C%E7%B4%A2%E5%BC%95%E6%93%8E%E5%88%87%E6%8D%A2%E5%99%A8).

### Installation & Usage
1. Install a userscript manager (Tampermonkey, Violentmonkey, or Greasemonkey) in your browser.
2. Click the script's **Install** button, or import the `.user.js` code into your manager.
3. Visit any supported search page; the floating search engine switcher will appear automatically.
4. Click the **gear icon** to manage built-in and custom search engines.
5. Click **➕ Add Current Site** to quickly save the current search page as a custom engine.

### Notes
- The script automatically detects many popular search engines, but you can manually add any site that supports query parameters.
- Custom engines are saved locally and persist across page loads.
